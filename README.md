# AWS Chooser


AWS Chooser is a browser extension designed to help users quickly select and switch between multiple AWS accounts and services. It provides a simple popup interface to manage AWS SSO URLs, configure account and service lists, and easily copy or navigate to the desired AWS console. The extension is ideal for developers, admins, or anyone who frequently works with multiple AWS environments.

**Features:**
- Configure and edit lists of AWS accounts and services
- Quickly search and select accounts/services from a popup
- Simple settings page for easy configuration

**Todos**
- Use arrows for navigation
- Add action to go back to accounts
- remove focus from extension after action
